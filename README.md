# Hospital Management System (Flask + SQLite)

## Overview
A simple hospital management system built with Flask and SQLite. Supports management of Patients, Doctors, and Appointments with basic CRUD operations and a minimal Bootstrap UI.

## Setup
1. Create and activate a Python virtual environment.
2. `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Open `http://127.0.0.1:5000` in your browser.
